RESUMEN

Este es un juego ambientado en el espacio en tercera persona. El objetivo es controlar tus naves para destruir la base enemiga mientras esta hace lo mismo para ganarte.

El juego dispone de un menú principal en el que se puede elegir jugar, seleccionar nivel o salir de la aplicación. Hay 3 niveles: un tutorial para familiarizarse con los controles, un nivel normal y un nivel más dificil.

Se pueden seleccionar grupos de unidades y darles indicaciones, ya sean posiciones o unidades enemigas. Las unidades aliadas se encuadran en verde y las enemigas en rojo. En la partida hay una GUI que indica las unidades de cada equipo y la vida de su base.

------------------------------------------------------------

CONTROLES

Ratón:
* Arrastrar ratón izquierdo: seleccionar unidades dentro del recuadro o desseleccionar todas en caso se no haber nada.

-Si hay unidades seleccionadas:
* Clic derecho: indicar posición en el plano auxiliar (si se mantiene apretado se puede indicar una posición fuera del plano en vertical) o sobre otras unidades.
* LSHIFT + Arrastrar clic izquierdo: mover camara alrededor de las unidades seleccionadas.

-Si no hay unidades seleccionadas:
* Arrastrar ratón derecho: mover posición camara.

Teclas:
* A: Seleccionar todas las unidades.
* S: Alternar entre posicionamento en linea y en circulo al mover unidades.
* E: Centrar camara para enfocar unidades seleccionadas.
* ESC: Abrir menu.
* SPACE: Pausar juego. Permite mayor control de las unidades por parte del jugador.
* Flechas: Mover la camara. (+LSHIFT para mayor velocidad).
